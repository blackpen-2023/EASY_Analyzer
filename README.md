## EASY ANALYZER 2024
릴스를 넘기며 우연히 입력된 값을 다시 리턴해주는 분석기를 보고 영감을 얻어 제작하게 되었습니다.
사용자에게 내용을 입력받고, 입력받은 내용이 무엇인지 알려주는 분석기 입니다.
즉 '사과'를 입력하게 되면 입력받은 내용이 '사과'임을 알려줍니다.
블록체인과 관련된 내용은 그냥 넣었습니다. 아무런 관련 없습니다. ^^7
